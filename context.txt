🔹 Step 3: Add Files

Your repo should look like this:

my-ml-model-space/
│── app.py        # main app code
│── requirements.txt

🔹 Step 4: Write app.py (Gradio Example)
import gradio as gr
import joblib

# Load model
model = joblib.load("model.pkl")

# Define prediction function
def predict(features):
    # features is a list input from UI
    prediction = model.predict([features])
    return str(prediction[0])

# Gradio interface
iface = gr.Interface(
    fn=predict,
    inputs=[gr.Textbox(label="Enter features (comma separated)")],
    outputs="text",
    title="My ML Model"
)

if __name__ == "__main__":
    iface.launch()

🔹 Step 5: Add Dependencies

Create a requirements.txt:

gradio
scikit-learn
joblib


(Add TensorFlow, PyTorch, etc. if your model needs them.)

🔹 Step 6: Push to Hugging Face

In terminal:

git clone https://huggingface.co/spaces/username/my-ml-model
cd my-ml-model
# Copy your files into this folder
git add .
git commit -m "Initial commit"
git push


Once pushed, Hugging Face will auto-build your Space.
✅ You’ll get:

A UI where you can test your model.

A REST API endpoint like:

https://username-my-ml-model.hf.space/run/predict

🔹 Step 7: Use as an API (optional)

From Python, you can call it:

import requests

resp = requests.post(
    "https://username-my-ml-model.hf.space/run/predict",
    json={"data": [[1, 2, 3, 4]]}  # Example features
)
print(resp.json())
